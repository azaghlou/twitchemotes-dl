# twitchemotes-dl
Grab emotes from twitchemotes.com

## Usage :

1. Clone repo

2. Install deps `npm i`

3. Get channel `[id]` from `https://twitchemotes.com/channels/[id]`

4. Use something like `node ./scrap.js [id]`
